# TSL Toolbox Releases

Public download page for the **TSL Toolbox** browser extension.

| Use this repo for | Do not use this repo for |
|-------------------|--------------------------|
| [Downloading releases](https://github.com/sebbonit/tsl-toolbox-releases/releases) | Source code (private) |
| [Reporting bugs or support issues](https://github.com/sebbonit/tsl-toolbox-releases/issues) | Pull requests or code contributions |

## Install

1. Open [Releases](https://github.com/sebbonit/tsl-toolbox-releases/releases)
2. Download the latest Chrome or Firefox zip for your browser
3. Install the extension using your browser's developer / sideload flow

## Support

Open an [issue](https://github.com/sebbonit/tsl-toolbox-releases/issues/new/choose) for bugs, install problems, or questions about a release.

You can also report bugs from inside the extension.

## Contributions

This repository is **not** open for contributions. Pull requests will be closed automatically.

The extension source code is private. These releases are provided for installation only.
